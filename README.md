[모두의 스터디]
기획: 황보현우 / 제작 협력: Gemini (Google AI)

누구나 쉽게 자신만의 모의고사를 제작하고 공유할 수 있는 오픈소스 프로그램입니다.

---

1. 프로그램 소개
본 프로그램은 각종 자격증 시험 및 암기 과목 공부를 위해 제작되었습니다.
직관적인 UI를 통해 문제를 관리하고, 실제 시험과 유사한 환경에서 모의고사를 치를 수 있습니다.

2. 주요 기능 설명
- 과목별 폴더 관리: 공부하고 싶은 주제별로 폴더를 생성하여 체계적으로 관리할 수 있습니다.
- 무작위 문제 출제: 매 시험마다 문제 순서가 무작위로 섞여 나와 답의 순서를 외우는 것을 방지합니다.
- 회차별 오답 노트: 틀린 문제에 대해 [질문 / 내가 쓴 답 / 실제 정답]을 기록하며, 회차별 독립 파일로 저장됩니다.
- 시험 통계 시스템: 최근 시험 기록을 백분율(%) 점수와 함께 확인 가능합니다.
- 중도 하차 점수제: 시험 도중 중단하더라도 현재까지의 정답률을 계산하여 성적을 기록해 줍니다.
- 데이터 백업 및 공유: 과목 전체를 .zip 파일로 내보내거나 타인의 과목 파일을 불러올 수 있습니다.

3. 설치 및 실행 방법
- 배포된 압축 파일을 적절한 위치에 해제합니다.
- '모두의스터디.exe' 파일을 실행합니다.

4. 라이선스 및 배포 원칙
- 본 프로그램은 모든 사용자에게 무료로 제공되는 오픈소스 소프트웨어입니다.
- 사용자는 자유롭게 프로그램을 수정, 업그레이드 및 재배포할 수 있습니다.
- 단, 본 프로그램 및 수정본을 유료로 판매하거나 상업적인 목적으로 이용하는 행위는 엄격히 금지됩니다.
